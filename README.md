# myblog
springboot+mybatis-plus 博客系统

项目包含博客展示+博客管理

**技术组合：**

*  后端：Spring Boot + Mybatis-Plus + thymeleaf模板
*  数据库：MySQL
*  前端UI：Semantic UI框架
